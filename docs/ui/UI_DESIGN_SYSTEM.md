# UI Design System

- Collapsed sidebar: 72 px
- Expanded sidebar: 260 px
- Hover animation: 220–280 ms
- Card radius: 18–22 px
- Primary: #6C5CE7
- Blue: #4F7CFF
- Cyan: #42D3FF
- Pink: #F45CB4
- Background: #EEF3FF

## Phase 3 shell

- Qt Widgets only
- Light glass-effect workspace with translucent white surfaces
- Purple-blue-cyan code-painted icons
- Rounded 20 px cards with soft shadows
- Sidebar tooltips in the collapsed state
- Active navigation uses a subtle gradient highlight
- Dashboard and Settings remain presentation-only placeholders

## Phase 5 dashboard

- Six service-backed KPI cards with typed zero states
- Date filters for today, seven days, and thirty days
- Production pipeline, recent activity, and low-stock panels
- Quick actions are limited to refresh and existing Settings navigation
- Dashboard UI imports no database or ORM modules

## Phase 6 customers

- Searchable and status-filtered customer table
- Glass-styled create/edit form with billing and shipping tabs
- Read-only customer details dialog
- Service-backed deactivation rather than destructive deletion
